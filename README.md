# vue-template
一个基于webpack4+vue运行环境
## 起步
### 通过脚手架安装
    ```
        npm install devil-cli -g //安装脚手架
        devil init template name //初始化项目，name为你初始化项目的名称
    ```
### 直接克隆
    ```
        git clone https://github.com/ilzzsthl/vue-template.git
    ```
## 运行
    ```
        npm install
        npm start
    ```
## 注意
目前版本只供学习使用，请勿使用于生产环境。
